# Week 3 Workshop Code — Data Structures & Comprehensions

## Overview
This repository contains my Week 3 Python workshop exercises from the ITEC102 course. The notebook builds on previous concepts and introduces more advanced data structures and efficient coding techniques using comprehensions.

The goal of this workshop was to strengthen my understanding of how Python stores, manipulates, and processes data using lists, tuples, sets, and dictionaries, while also improving code efficiency through comprehension techniques.

---

## Concepts Demonstrated
This notebook includes examples of:
- Lists (creation, modification, sorting, and deletion)
- Tuples and accessing structured data
- Sets for removing duplicates and performing operations
- Dictionaries for storing and updating key-value data
- List, set, and dictionary comprehensions
- Filtering and transforming data using conditions
- Iterating through collections using loops

Exercises in this workshop include:
- Managing and modifying lists (e.g., suburbs and foods)
- Using sets to remove duplicate values and find intersections
- Creating and updating dictionaries for structured data storage
- Filtering data from lists of dictionaries (e.g., open vs closed cafés)
- Generating lists, sets, and dictionaries using comprehensions
- Applying conditions within comprehensions for filtering data

---

## Mini Project - Student Enrolment System

This workshop also includes a small project simulating a student workshop enrolment system.

Features of this system include:
- Storing student data using tuples
- Allowing users to select workshops via input
- Validating selections using sets (removing duplicates and ensuring valid entries)
- Using dictionaries to map student enrolments
- Dynamically adding new students and workshops
- Displaying a formatted enrolment summary
- Calculating workshop popularity using dictionary aggregation

This project demonstrates how multiple Python concepts can be combined to create a simple but functional system.

## Running the Notebook

### Option 1 - Google Colab (Recommended)

1. Open the notebook in Google Colab
2. Run each cell in order

### Option 2 - Jupyter Notebook

1. Install Jupyter Notebook if not already installed:
```
pip install notebook
```
2. Launch Jupyter Notebook:
```
jupyter notebook
```
3. Open the notebook and run each cell in order.
